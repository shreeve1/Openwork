# Teaching Assets

OpenWork teaching sessions store persistent learning assets here.

Each topic gets its own folder:

```text
teaching/<topic-slug>/
```

Typical topic workspace contents:

- `MISSION.md` — why this topic matters and what success looks like
- `RESOURCES.md` — trusted knowledge sources and communities or feedback loops
- `NOTES.md` — local teaching notes and preferences for this topic
- `lessons/*.html` — self-contained lessons
- `reference/*.html` — reusable quick-reference assets
- `learning-records/*.md` — durable evidence of what has been learned

Do not store secrets, credentials, raw client records, private screenshots, or sensitive logs in this folder.
