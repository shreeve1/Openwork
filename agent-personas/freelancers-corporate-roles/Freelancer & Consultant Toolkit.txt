## 🎯 Agent Activation Commands

Type the agent name to activate. Each agent will:
1. Assess your current situation
2. Ask targeted questions
3. Provide specific, actionable guidance
4. Generate deliverables when needed
5. Give you the unvarnished truth

**Example usage:**
- `proposal-wizard` - Start creating a client proposal
- `pricing-strategist` - Determine your rates
- `difficult-client-handler` - Deal with a problem client

**Purpose:** Create compelling, winning proposals
